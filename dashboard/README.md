# DRL CloudSim Dashboard

This is a minimal React + Vite scaffold for the DRL CloudSim dashboard.

Quick start:

```powershell
cd "d:\Major Project\dashboard"
npm install
npm run dev
```

The frontend expects a WebSocket bridge at `http://localhost:4000` (Socket.IO). I can add a small Node.js bridge or update the Java/Python code to use WebSockets next.
